# DailyExpenseTracker

A full-stack Daily Expense Tracker application to help users record, manage, and analyze their daily expenses with ease.  
This project includes both **backend** and **frontend** components to provide a seamless user experience and efficient data handling.

## Features

1. Add, edit, and delete expense records  
2. Track expenses by date, category, and description  
3. View summaries and totals of your expenses  
4. Responsive and user-friendly interface  
5. RESTful API backend to support CRUD operations



##  Technologies Used

- **Frontend:** React 
- **Backend:** Django
- **Database:** SQLite
- **API:** RESTful endpoints for expenses management
